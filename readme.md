# RPG Game with Ebiten

![Game Screenshot](readme/screenshot.png)

## Description

This is a basic RPG game built with [Ebiten](https://ebitengine.org/), a 2D game engine for Go. The game is still in its early stages. 

## Controls

- AWSD - movement
- shift - dashing
- mousewheel - zooming
